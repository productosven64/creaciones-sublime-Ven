# Creaciones Sublime

Sitio estático para Creaciones Sublime — catálogo de franelas, tazas, aluminio y souvenirs.

Estructura principal:
- `index.html` — página principal
- `style.css` — estilos
- `main.js` — scripts
- `assets/` — imágenes y recursos

Cómo subir al remoto (si ya tienes creado el repo en GitHub):

```bash
# desde la carpeta del proyecto
git remote add origin https://github.com/<tu-usuario>/<tu-repo>.git
git branch -M main
git push -u origin main
```

Si quieres, puedo intentar el `git push` por ti si me das la URL válida del repo o un token con permisos (no recomendado compartir tokens aquí). Alternativamente puedo generar un ZIP del proyecto para que lo subas manualmente.
